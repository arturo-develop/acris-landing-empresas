# Acris IA - Landing Empresas

Sistema de ventas con IA para empresas. Landing page optimizada con visualización interactiva de canales integrados.

## Características

- Diseño responsive mobile-first
- Visualización animada del cerebro IA con canales conectados
- Sistema de glassmorphism y gradientes
- Optimizado para conversión
- WhatsApp, Instagram, Email, Web, Llamadas, CRM integrados

## Deploy

Este proyecto está listo para deployarse en cualquier hosting estático:
- Vercel
- Netlify
- GitHub Pages
- EasyPanel

## Estructura

```
├── index.html          # Landing page principal
└── README.md          # Documentación
```

## Tecnologías

- HTML5
- CSS3 (Variables custom, animations, glassmorphism)
- Vanilla JavaScript
- Google Fonts (Space Grotesk, Inter)

## Desarrollo local

Simplemente abre `index.html` en tu navegador o usa un servidor local:

```bash
# Con Python
python -m http.server 8000

# Con Node.js
npx serve
```
